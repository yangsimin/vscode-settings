| 命令 | Mac | Windows |
| :----- | :------ | :----- |
| 显示建议 | cmd + i | |
| 显示参数提示 | cmd + shift + sapnce | |
| 显示 code action | cmd + .| |

| 选择所有当前单词 | cmd + F2 | |
